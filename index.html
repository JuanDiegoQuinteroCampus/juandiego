Etiquetas de apertura y cierre de una tabla. El resto de etiquetas de la tabla han de ir siempre recogidas entre estas dos etiquetas.
Usada para indicar el título de la tabla. Etiqueta utilizada para agrupar dos o más columnas de una tabla. Usada para describir los datos concretos de una tabla. Indica el bloque de filas que describen las etiquetas de las columnas de la tabla. Indica los bloques de filas que describen los resúmenes, o datos totales de una columna de una tabla. Se usa para indicar una fila de celdas de una tabla. Usada para definir una celda de una tabla. Etiqueta que se usa para definir el encabezado de una celda
